# AutoVideo

An AI-powered system that transforms quotes from Murphy’s Law into engaging short-form videos. It automatically breaks content into micro-scenes, generates visuals, adds voice narration, and stitches clips into dynamic reels optimized for YouTube Shorts, enabling scalable, viral-ready content creation.

**Chạy API (FastAPI):** `pip install -r requirements.txt && pip install -e .` rồi `python app.py` hoặc `uvicorn api.app:app --host 0.0.0.0 --port 8000` — mở `/docs`. Chi tiết: [IMPLEMENTATION.md](IMPLEMENTATION.md).
